## local setup

```
# Create virtual environment
python3 -m venv .venv

# Activate virtual environment
source .venv/bin/activate

# Create requirements.txt with necessary packages
cat > requirements.txt << EOL
pandas
cassandra-driver
numpy
EOL

# Install requirements
pip install -r requirements.txt
```
